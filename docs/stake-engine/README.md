# StakeEngine Documentation

This directory contains local copies of the StakeEngine documentation for reference by GitHub Copilot and development tools.

## Documentation Structure

The documentation is organized into the following sections:

- **getting-started/** - Introduction, RGS details, wallet endpoints, and basic examples
- **front-end/** - Front-end development guides, dependencies, and UI implementation
- **math/** - Game mathematics, probability calculations, and verification
- **approval-guidelines/** - Platform compliance and quality requirements
- **legal/** - Terms, conditions, and legal requirements

## Usage

This documentation is referenced by GitHub Copilot through the `.github/copilot/instructions.md` file. When working on StakeEngine projects, Copilot will consult these local files for accurate, context-aware assistance.

## Maintenance

Documentation content should be synchronized with the official StakeEngine documentation at https://stake-engine.com/docs when updates are available.
